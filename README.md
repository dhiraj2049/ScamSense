# Review Shield — Fake Product Review Detector

Online reviews influence what people buy, but not every review is honest.

Some are overly promotional, misleading, or completely fake.

**Review Shield** is an AI-powered web application built to detect suspicious product reviews and help users make smarter decisions.

> Is this review genuine or trying to sell you something?

---

## What This Project Does

Review Shield analyzes review text and predicts whether it looks:

* Genuine
* Suspicious
* Spam-like
* Overly Promotional

It studies writing patterns, repeated phrases, extreme sentiment, and common fake-review behavior.

---

## Core Features

### Instant Review Check

Paste any review and get:

* Fake vs Genuine prediction
* Confidence score
* Risk level

### Bulk CSV Upload

Upload multiple reviews at once for batch analysis.

Useful for businesses, marketplaces, and researchers.

### Visual Insights

See charts showing:

* Genuine review percentage
* Suspicious review percentage
* Overall trust level of the dataset

### Smart Text Understanding

The system detects patterns such as:

* “Best product ever!!!”
* “Must buy now”
* “Limited time offer”
* “Amazing amazing amazing”

## How It Works

Review Shield converts review text into data the model can understand.

It looks at:

* Important keywords
* Common spam phrases
* Repeated language
* Tone exaggeration
* Word combinations

Then it predicts whether the review appears trustworthy or suspicious.

---

## Why This Project Matters

Fake reviews can:

* Mislead buyers
* Hurt honest sellers
* Damage trust in platforms
* Influence bad purchases

Review Shield helps reduce that problem using AI.
